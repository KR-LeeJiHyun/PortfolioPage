# 프로젝트 요약
커뮤니티 게시판을 구축해보자!!!
Spring과 jsp를 활용한 커뮤니티 게시판보드이다.

# SeoLee 목표
- 회원가입을 한다.
- 회원정보 수정이 가능하다.
- 로그인 및 유지 기능을 제공한다.
- 게시글을 쓰기, 삭제, 수정, 조회가 가능하다.
- 댓글을 쓰기, 삭제, 수정, 조회가 가능하다..

# 기술 스택
Spring / JSP / JSTL / Oracle / Tomcat

# 팀원
2명
저는 본 프로젝트에서, 공통된 부분이외에 게시글을 담당했습니다.
